
Introduction
============

Proof of concept for mr.migrator.

Installation
============

Follow these steps::

    $ git clone git@github.com:aclark4life/mr_migrator_demo.git
    $ cd mr_migrator_demo
    $ python2.7 bootstrap.py -d
    $ bin/buildout
    $ bin/plone start
    $ bin/migrate

You should end up with content in Plone that looks like this:
